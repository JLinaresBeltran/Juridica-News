Nueva Ley 4/2024 de protección de datos en el ámbito sanitario

La nueva regulación establece medidas específicas para la protección de datos personales en el sector sanitario, incluyendo tratamientos especiales para datos de salud, requisitos de consentimiento informado y protocolos de seguridad obligatorios para centros médicos.

Nuevas obligaciones fiscales para empresas de economía digital
